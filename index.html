Next Best Offer (NBO) Recommender System

Designed and implemented a centralized NBO system for a major bank to deliver automated, customer-centric product offers. The framework combined customer similarity, product similarity, and XGBoost propensity modeling, tested through leave-one-out. An ensemble approach was also explored, showing marginal improvements over the strongest models.

Role:

  • Pipeline design
  • Model development
  • Evaluation
  • Manuscript and presentation

Tech Stack:

  • Python: NumPy, Pandas, Matplotlib, Scikit-learn, XGBoost

Note: Due to NDA restrictions, source code and client data cannot be shared. A synthetic demo and abstract are provided instead.

Using Topology to Extract Insights from UAAP Basketball Data

This project explores the application of Topological Data Analysis (TDA) to identify groupings among players in the 84th season of the UAAP basketball tournament. We leverage the power of giotto-tda, a Python library specifically designed for TDA tasks. giotto-tda streamlines our workflow with its efficient functionalities, offers an intuitive interface for exploration, and benefits from the reliability of a well-established library within the TDA field.

Role:

  • Data collection
  • Pipeline design
  • Model development
  • Analysis
  • Manuscript & presentation

Tech Stack:

  • Python: Giotto-TDA, NumPy, Pandas, Matplotlib, Scikit-learn
